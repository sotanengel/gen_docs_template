# gen_docs_template

> **This CLI automatically creates templates of comment text required when generating docs.rs!!(docs.rs の生成時に必要なコメント文のテンプレートを自動作成する CLI です!!)**

# How to use

## install

Run the following command:

```
cargo install gen_docs_template
```
